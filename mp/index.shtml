MP Awareness Effort

In September 2006, several members of the GOSLING community of practice took part in an outreach project. The focus of this project was to provide Members of Parliament with information about global FLOSS communities. The package also included CDs containing FLOSS software. In order to share their work and encourage similar efforts, the project participants have made available the source of the package contents written by the group.

Community Authored Materials

These files are available in a number of formats for convenience (OpenOffice, PDF and Microsoft Word). All work has been released to the Public Domain.

  • Main insert explaining the project and setting context (English: odt, pdf, doc; French odt, pdf, doc)
  • Pamphlet introducing the GOSLING community (Bilingual: odt, pdf)
  • OpenOffice Fact Sheet (English: odt)
