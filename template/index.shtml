This site makes use of Server Side Includes (Both based on Apache's mod_include as well as PHP3's include) to put common headers/footers on every page. This header/footer is maintained in a central place, and changes to it show on all pages on the site.

The two methods are demonstrated as follows:

  • SSI Sample (Source)
    This is the version that should be used on most pages, and is the most portable and fastest for the server to process.

  • PHP Sample (Source)
    This one is used by the Software developers for interactive pages.

Accessability is something that is important for general web work, and as promoters of freedom of choice in other areas likely even more important for Green Party work. As a primer to doing pages, please read the Accessible Design Guide. Once you page is done, please see if there are any critical problems with it by checking it with both the W3.org validator as well as Bobby.
